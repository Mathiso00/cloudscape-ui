## Cloudscape Design System

### Installation

```bash
pnpm add -D @koops/csui
```

### Usage

```tsx
import { Dialog } from '@koops/csui'

function App() {
  return <Dialog>Click me</Dialog>
}
```

### Development

```bash
pnpm install
pnpm dev
```

### Build

```bash
# With watch mode
pnpm build
```
