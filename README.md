## Cloudscape Design System

### Installation

```bash
pnpm add -D @koops/csui
```

### Usage

```tsx
import { Button } from '@koops/csui'

function App() {
  return <Button>Click me</Button>
}
```

### Development

```bash
pnpm install
pnpm dev
```

### Build

```bash
# With watch mode
pnpm build
```
